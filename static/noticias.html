Bocetos de noticias

Las noticias las podrá ver cualquier persona, los organizadores y administradores podrán publicar noticias, para acceder hay que darle a noticias.

Esta es la pantalla principal de las noticias, se podrá ver el titular de todas las noticias publicadas.

Así es como lo verían los distintos usuarios:

Usuario sin registrar

Usuario registrado

Organizador o administrador

Al seleccionar una noticia se podrá ver la noticia entera, se verá así para los distintos usuarios:

Usuario sin registrar

Usuario registrado

Organizador o administrador

Los organizadores y administradores podrán crear noticias, deberán darle a crear noticia.

Después de escribir la noticia deberán darle a publicar.
